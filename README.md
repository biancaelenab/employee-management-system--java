## Employee Management System App
Console application written in Java for managing employee records.
The program allows the user to add, update, delete, and search employees using a simple interactive menu.


# Features
- Add a new employee  
- Update employee information  
- Delete an employee  
- Search employees by ID  
- Display all employees
  
# How to Run
1. Clone the repository  
2. Open the project in an IDE (e.g., IntelliJ IDEA or Eclipse)  
3. Run the `Main` class  
4. Follow the menu instructions in the console  

# Project Structure
- `Employee` – model class representing employee data  
- `EmployeeService` – contains business logic and CRUD operations  
- `Main` – handles user input and menu navigation  
- Input validation  
- Better error handling  
- GUI or web interface
